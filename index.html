บทที่ 1 · VS Code คืออะไร เริ่มต้น

เข้าใจว่า VS Code คือเครื่องมืออะไร และทำไมโปรแกรมเมอร์ทั่วโลกเลือกใช้

Visual Studio Code (VS Code) คือโปรแกรมแก้ไขโค้ด (Code Editor) ฟรีจาก Microsoft ทำงานได้ทั้ง Windows, macOS และ Linux รองรับภาษาโปรแกรมแทบทุกภาษา มีระบบส่วนเสริม (Extensions) จำนวนมาก ทำให้ปรับแต่งได้ตามต้องการ

จุดเด่นที่ควรรู้

  • เบา เปิดเร็ว ใช้ทรัพยากรน้อย
  • มี IntelliSense ช่วยเติมโค้ดอัตโนมัติ
  • มี Terminal และ Git ในตัว
  • มี Extensions หลายหมื่นตัวให้เลือก
💡 VS Code ไม่ใช่ Visual Studio (IDE ของ Microsoft) — เป็นคนละโปรแกรมกัน

บทที่ 2 · ติดตั้งและเปิดใช้งานครั้งแรก

ดาวน์โหลด ติดตั้ง และตั้งค่าภาษาไทย
  1. ไปที่เว็บไซต์ https://code.visualstudio.com
  2. กดปุ่ม Download ตามระบบปฏิบัติการของคุณ
  3. ติดตั้งโดยกด Next ไปจนเสร็จ (แนะนำให้ติ๊ก “Add to PATH”)
  4. เปิดโปรแกรม จะเห็นหน้า Welcome

เปลี่ยนเป็นภาษาไทย (ออปชัน)

  1. กด Ctrl+Shift+P เปิด Command Palette
  2. พิมพ์ Configure Display Language
  3. เลือก Install additional languages... แล้วเลือก Thai
  4. กด Restart

ส่วนที่ใช้ดูไฟล์ในโฟลเดอร์โปรเจกต์เรียกว่าอะไร?

บทที่ 4 · การเปิดไฟล์และโฟลเดอร์

  • เปิดไฟล์เดี่ยว: File > Open File หรือ Ctrl+O
  • เปิดทั้งโฟลเดอร์ (แนะนำ): File > Open Folder หรือ Ctrl+K Ctrl+O
  • สร้างไฟล์ใหม่ใน Explorer: คลิกขวา > New File
  • บันทึก: Ctrl+S · บันทึกทั้งหมด: Ctrl+K S
⚠️ ควรเปิดเป็น “โฟลเดอร์โปรเจกต์” เสมอ เพื่อให้ Git, Extensions และการค้นหาทำงานได้ครบ

คีย์ลัดเลือกคำเดียวกันถัดไปเพื่อแก้พร้อมกันคือ?

บทที่ 6 · ค้นหาและแทนที่

  • ค้นหาในไฟล์: Ctrl+F
  • แทนที่ในไฟล์: Ctrl+H
  • ค้นหาทั้งโปรเจกต์: Ctrl+Shift+F
  • ไปที่ไฟล์อย่างรวดเร็ว: Ctrl+P
  • ไปบรรทัดที่ต้องการ: Ctrl+G
💡 ในกล่องค้นหา กดปุ่ม .* เพื่อใช้ Regex, Aa สำหรับ case-sensitive

บทที่ 7 · Extensions ที่ควรติดตั้ง

เปิดมุมมอง Extensions ด้วย Ctrl+Shift+X

  • Prettier — จัดรูปแบบโค้ดอัตโนมัติ
  • ESLint — ตรวจ error สำหรับ JavaScript/TypeScript
  • Live Server — เปิดเว็บ HTML แบบ auto-reload
  • GitLens — ดูประวัติ Git ละเอียดขึ้น
  • Path Intellisense — เติม path ไฟล์อัตโนมัติ
  • Material Icon Theme — เปลี่ยนไอคอนไฟล์ให้สวยงาม

Extension ใดบ้างที่ใช้สำหรับ จัดรูปแบบโค้ด และ ตรวจ error ของ JavaScript?

บทที่ 8 · Terminal ในตัว

  • เปิด/ปิด Terminal: Ctrl+` (แป้นข้างเลข 1)
  • เปิดหลาย Terminal: กดปุ่ม + ที่มุมขวาของ Panel
  • เปลี่ยน shell ได้: PowerShell / Cmd / Git Bash / WSL

ตัวอย่างคำสั่งพื้นฐาน:

node -v        # ตรวจสอบเวอร์ชัน Node.js
npm init -y    # สร้างไฟล์ package.json
npm install    # ติดตั้ง dependencies
git status     # ดูสถานะ Git

บทที่ 9 · ใช้ Git เบื้องต้นใน VS Code

  1. ติดตั้ง Git ที่ https://git-scm.com
  2. เปิดโฟลเดอร์โปรเจกต์ใน VS Code
  3. คลิกไอคอน Source Control (รูปแยกกิ่ง) ที่ Activity Bar
  4. กด Initialize Repository
  5. เขียน commit message แล้วกด ✓ เพื่อ commit

คำสั่ง Git ใน Terminal ที่ใช้บ่อย:

git init
git add .
git commit -m "first commit"
git remote add origin <URL>
git push -u origin main

🧪 แบบฝึกหัด 9 — เติมคำสั่ง

คำสั่งสำหรับ “บันทึกการเปลี่ยนแปลง” พร้อมข้อความ คือ git ____ -m "msg"

บทที่ 10 · Debug เบื้องต้น

  • คลิกไอคอน Run and Debug ที่ Activity Bar (หรือ Ctrl+Shift+D)
  • คลิกซ้ายข้างเลขบรรทัด เพื่อตั้ง Breakpoint (จุดสีแดง)
  • กด F5 เพื่อเริ่ม Debug
  • ปุ่มควบคุมหลัก: Continue (F5), Step Over (F10), Step Into (F11)
💡 ใช้ Debug Console พิมพ์ตัวแปรเพื่อดูค่าระหว่าง debug ได้เลย
